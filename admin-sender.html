⚙️ One-time setup — takes 5 minutes
  1. Go to console.firebase.google.com → Create a project (free)
  2. Add a Web App → copy the firebaseConfig object
  3. Go to Project Settings → Cloud Messaging → Web Push Certificates → Generate Key Pair → copy the VAPID public key
  4. Paste everything into the Config card below and into your protipsters.html file at the FCM_CONFIG section
  5. For the Server Key (to send blasts): Project Settings → Cloud Messaging → Server Key
